# Smart Task Manager with Analytics

A modern **full-stack task management web app** built with **Flask, SQLite, HTML, CSS, and Pandas**. This project allows users to create, edit, complete, delete, and track tasks while providing a **dashboard and analytics** for productivity insights.

---

## **Features**

- ✅ **Add, Edit, Complete, Delete Tasks**  
- 📊 **Dashboard** showing total, pending, and completed tasks  
- 📈 **Analytics Page** with charts for task status and due dates  
- 🔍 **Search Tasks** by title  
- 🎨 **Priority Levels** (Low, Medium, High)  
- 💻 **Responsive, Modern UI** with hover tooltips on action buttons  
- 🗄 **SQLite Database** for storing all tasks  

---


## **Technologies Used**

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Frontend:** HTML, CSS, JavaScript  
- **Data Analysis:** Pandas  
- **Charts:** Chart.js (for analytics)  

---

How to run
1. Install dependencies: pip install -r requirements.txt or pip install flask pandas
2. Run python app.py
